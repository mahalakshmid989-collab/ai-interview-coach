# 🎓 AI Interview Coach

A Flask-based web application that simulates HR and Technical interviews with real-time feedback and scoring.

## 🚀 Features

* HR & Technical interview modes
* Random non-repeating questions
* Answer evaluation system
* Score tracking
* PDF report generation

## 🌍 Live Demo

https://your-app.onrender.com

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS
* ReportLab (PDF generation)

## ▶️ Run Locally

```bash
git clone https://github.com/your-username/ai-interview-coach.git
cd interview-coach
pip install -r requirements.txt
python app.py
```
